# BSUIR_POIT_DateChecker

## Описание
Проект **BSUIR_POIT_DateChecker** — это простое приложение с графическим интерфейсом, разработанное на Python с использованием библиотеки Tkinter. Оно предназначено для определения времени года по введенной пользователем дате в формате ДД.ММ.

## Установка

1. Убедитесь, что у вас установлен Python (версия 3.x).
2. Установите необходимые зависимости, используя pip:
   ```bash
   pip install tk
   pip install pyinstaller
   ```

## Создание исполняемого файла (EXE)

Чтобы создать исполняемый файл из проекта, выполните следующую команду в терминале или командной строке:

```bash
pyinstaller --onefile --windowed .\main.py
```

### Параметры команды:
- `--onefile`: создаёт один исполняемый файл.
- `--windowed`: отключает консольное окно, что делает приложение более подходящим для GUI.

После выполнения команды в каталоге вашего проекта будет создана папка `dist`, в которой вы найдете ваш исполняемый файл `main.exe`.
